## How to Use

### Step 1: Go to Admin
```
http://127.0.0.1:8000/admin/bank/llmprompt/
```

### Step 2: Add New Prompt
Click **"Add LLM Prompt"** button

### Step 3: Fill In
```
Source URL: https://example-news.com/news
Prompt Type: MCQ (or Descriptive)
Prompt Text: [Your custom prompt with {title} and {content}]
Is Default: [unchecked - since it's source-specific]
Is Active: [checked]
```

### Step 4: Save
Click **"Save"** - Done!
